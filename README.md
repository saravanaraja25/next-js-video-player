# Video Player

This is a simple video player developed using Next.js, Tailwind CSS, and Shadcn.

## Features

- Play/Pause
- Volume Control
- Fullscreen
- Seek
- Picture-in-Picture
- Video Quality
- Video Speed
- Video Thumbnail
- Video Title
- Video Description
- Video Duration
- Video Progress
- Video Resume from Last Played Position
